# Hamza Hasan Islamic Foundation Website

A fully functional, modern Islamic organization website built with React and Vite. The website showcases comprehensive services, events, team information, and community engagement features.

### 🌟 Features

### Core Pages
- **Home**: Hero section, feature highlights, statistics, upcoming events, and call-to-action
- **About**: Organization history, mission, vision, core values, founder information, and achievements
- **Services**: Comprehensive list of 8+ services including Qur'an classes, Islamic education, youth programs, and community welfare
- **Events**: Event listing with category filtering, event details, and newsletter subscription
- **Team**: Leadership team profiles, volunteer opportunities, and career inquiries
- **Donate**: Multiple donation options, program selection, recurring donations, and tax information
- **Contact**: Contact information, contact form, office hours, social links, map, and FAQ

### Technical Features
- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Client-side routing with React Router
- ✅ Modern gradient UI with smooth animations
- ✅ Interactive forms and filtering
- ✅ Professional styling with CSS
- ✅ Accessibility-friendly components
- ✅ Fast performance with Vite build tool

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to:
```
http://localhost:5173/
```

## 📦 Dependencies

- **react**: ^18.x - UI library
- **react-dom**: ^18.x - React DOM rendering
- **react-router-dom**: ^6.x - Client-side routing
- **vite**: ^8.x - Build tool and dev server
- **axios**: Http client (for future API integration)

## 💡 Customization

### Adding New Pages
1. Create a new file in `src/pages/`
2. Create corresponding CSS in `src/styles/`
3. Add the route to `App.jsx`
4. Update navigation menu

### Modifying Colors
Edit color variables in:
- `src/styles/Navigation.css`
- `src/styles/Footer.css`
- Individual page CSS files
